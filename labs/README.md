# Course Labs

Three foundation labs that build the skills behind what Claude Code automates.

## How to use these skeletons

1. Copy the skeleton file to a working file: `cp labs/lab-X-skeleton/X_skeleton.py labs/lab-X-skeleton/X.py`
2. Fill in the TODOs
3. Run against the sample data in `labs/lab-X-samples/`
4. Compare your output to the expected output in the comments

## Minimum viable versions

Each lab has a minimum viable version (30-60 min) defined in `docs/course/lab-X-rag.md` etc. Start there before attempting the stretch goals.
